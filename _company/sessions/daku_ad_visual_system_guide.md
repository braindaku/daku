# 🎨 Daku 광고 디자인 시스템 가이드 (Visual System Guide)

## 1. 🎯 시스템 개요 및 목표
이 가이드는 Daku의 모든 SNS 광고(유튜브 썸네일, 인스타그램 피드, 스토리 등)에 일관성을 부여하고, 제작 효율성을 극대화하는 것을 목표로 합니다. 모든 크리에이티브는 이 시스템을 따릅니다.

## 2. 🎨 브랜드 디자인 시스템 (Brand DNA)

### 2.1. 컬러 시스템 (Color Palette)
| 용도 | 이름 | HEX Code | 역할 | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| **Primary** | Daku Blue | `#1A4A8D` | 브랜드 메인 색상, 헤더, 강조 요소 | 신뢰감, 전문성 |
| **Secondary** | Accent Yellow | `#FFC300` | CTA 버튼, 주의 유도, 하이라이트 | 활력, 주목도 |
| **Background** | Neutral Light | `#F9F9F9` | 배경, 여백 처리 | 깨끗함, 가독성 확보 |
| **Text/Dark** | Deep Gray | `#333333` | 본문 텍스트, 주요 제목 | 가독성 최적화 |
| **Warning** | Warning Red | `#CC3333` | 경고, 주의사항 (최소 사용) | 위기감 조성 |

### 2.2. 타이포그래피 시스템 (Typography)
*   **메인 폰트 (Headline/Title):** Pretendard (Bold, SemiBold)
    *   *사용 목적:* 제목, 핵심 키워드. 시선을 즉각적으로 사로잡는 힘이 필요할 때 사용.
    *   *규칙:* 폰트 크기 대비 여백(Leading)은 최소 1.2배를 유지.
*   **서브 폰트 (Body/Detail):** Noto Sans KR (Regular)
    *   *사용 목적:* 설명 문구, 본문 텍스트, 보조 정보. 가독성이 최우선일 때 사용.
    *   *규칙:* 행간(Line Height)은 최소 1.6배를 유지.

## 3. 🖼️ 레이아웃 및 그리드 시스템 (Grid & Layout)

### 3.1. 기본 그리드 원칙
*   **기준 단위:** 8pt Grid System을 기본 단위로 사용합니다. (여백, 간격, 크기 결정 시 8의 배수 사용)
*   **패딩/마진:** 모든 콘텐츠 블록은 상하좌우 최소 24px (3*8pt)의 여백을 확보하여 답답함을 해소합니다.

### 3.2. 필수 구성 요소 규격 (Component Specs)

#### A. CTA 버튼 (Call To Action)
*   **규격:** 가로 200px ~ 300px / 높이 48px
*   **색상:** Accent Yellow (`#FFC300`)
*   **텍스트:** "지금 확인하기" 등 명확한 행동 유도 문구.
*   **필수:** 버튼 주변에 최소 16px의 여백을 확보합니다.

#### B. 로고 배치 (Logo Placement)
*   **원칙:** 모든 광고의 우측 하단 또는 좌측 상단에 배치합니다.
*   **크기:** 가로 100px ~ 150px (광고 사이즈에 따라 가변적)
*   **주의:** 로고가 핵심 메시지나 CTA와 겹치지 않도록 레이아웃을 조정해야 합니다.

#### C. 썸네일/헤드라인 구성 (Visual Hierarchy)
*   **규칙:** 3초 안에 '무엇에 대한 콘텐츠'인지 인지할 수 있도록, **[후크 문구] > [주요 이미지/인물] > [브랜드 로고]** 순으로 정보를 배치합니다.
*   **후크 문구:** 전체 화면의 30%를 넘지 않도록 제한합니다. (최대 1~2줄)

## 4. 📱 사이즈별 적용 가이드 (Format Specific Guide)

### 4.1. 유튜브 썸네일 (1280 x 720 px)
*   **핵심:** 왼쪽 1/3 지점에 인물/핵심 이미지 배치 (클릭 유도)
*   **텍스트:** 후크 문구는 오른쪽 상단에 배치하고, Pretendard Bold를 사용합니다.
*   **규칙:** 오른쪽 하단에 로고 및 'Daku' 워터마크를 고정 배치합니다.

### 4.2. 인스타그램 피드 (1080 x 1080 px)
*   **핵심:** 균형 잡힌 텍스트와 이미지의 조화. 여백을 적극적으로 활용합니다.
*   **레이아웃:** 상단 1/3 (제목/핵심 질문) - 중단 1/3 (이미지/개념 설명) - 하단 1/3 (CTA/정보 출처).
*   **주의:** 텍스트 블록은 배경색(Neutral Light)을 사용하여 시각적 분리를 명확히 합니다.

### 4.3. 스토리/릴스 (1080 x 1920 px)
*   **핵심:** 몰입감과 흐름. 상하 여백을 최소화하고, 다음 단계로의 유도 장치(CTA)를 명확히 합니다.
*   **배치:** 텍스트를 화면 중앙에 집중시키고, CTA는 하단에 고정 배치합니다.
