## YouTube 채널 영상 분석 결과

**현재 분석 데이터:** 실시간 데이터 가져오기 실패.

**문제점:**
- YouTube API 키 또는 채널 ID가 설정되지 않았습니다.

**추천:**
- 위 메시지의 지시에 따라 `youtube_account.py` 파일에서 API 키와 채널 ID를 설정하세요.
- 설정한 후 다시 분석을 시도합니다.

**다음 단계:**
- 사용자가 `youtube_account.py` 파일에서 API 키와 채널 ID를 설정합니다.
- 분석 결과가 다시 생성됩니다.

**상기:**
- 추상적 조언 대신 **실행 가능한 산출물** (제목·썸네일 브리프·스크립트 후크)
- 매번 다음 단계 1줄을 명시
- 메모리(`memory.md`)에 누적된 댓글·반응 키워드를 후크에 반영
